A simple React app that lets users enter form details, preview them, and export the data as PDF or Excel files.

## Features
- Controlled React form with basic validation (required fields, email format)
- Read-only preview after successful submission
- Export as PDF using jsPDF and jspdf-autotable
- Export as Excel using SheetJS (xlsx) and FileSaver

## Setup
1. Install dependencies:

   npm install

2. Run development server:

   npm run dev

3. Build for production:

   npm run build

4. Preview build locally:

   npm run preview

## Notes
- Exports appear only after a successful form submission (preview visible).
- The project uses Vite + React.
